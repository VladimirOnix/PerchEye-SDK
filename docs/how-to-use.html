How to Use PerchEye SDK

Adding facial recognition to your Android app is simple with PerchEye SDK. Follow these three easy steps to get started.

1

Add the SDK to Your Project

Start by adding the PerchEye SDK to your Android project. Simply add the following to your app's build.gradle file:

// Add the repository
repositories {
    maven { url 'https://maven.percheye.com/releases' }
}

// Add the dependency
dependencies {
    implementation 'com.percheye.android:sdk:1.2.3'
}
2

Register Face for Future Verification

Use the SDK to scan a face and generate a unique identity hash. This hash can be stored in your app's database or preferences instead of the actual face image, providing better security and privacy.

The SDK stores a mathematical representation of facial features, not the actual image, ensuring privacy and data security.

3

Verify New Photos Against Stored Data

When a user wants to be verified, the SDK will compare their current face image against the previously stored hash and return a similarity score, making authentication simple and secure.

The SDK provides a simple similarity score (0-100%) that your app can use to make decisions about user authentication, access control, or personalized experiences.
